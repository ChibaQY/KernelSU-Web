# KernelSU Web

把 KernelSU Manager 的 UI 抠过来改了改，打开里面全是原神。

纯前端，零后端，所有按钮都能点，但点了也不会发生什么实质性的事情。

## 功能

- 首页仪表盘，两个主题可以切
- 超级用户列表，里面永远只有云·原神
- 模块列表，开关是假的但拨着挺解压
- 设置页，改啥都没用不过布局看着还挺像那么回事
- 安装页面、关于页面，该有的都有

## 使用

浏览器打开 index.html 就行，不需要装任何东西。

## 项目结构

KernelSU-Web/
├── index.html
