# KernelSU Web

把 KernelSU Manager 的 UI 扒下来改了改，打开里面全是原神。

所有按钮都能点，但点了也不会发生什么实质性的事情。taffy 也不知道为什么要做这个东西，反正做都做了喵。

## 功能

- 首页有两个主题可以切，切完还是那个样子
- 超级用户列表里永远只有云·原神，没有别的应用会出现了
- 模块列表的开关是真的可以拨的，但拨了也是假的
- 设置页面什么都有，改什么都不生效
- 剩下几个页面凑数的，该有的都有

## 怎么用

下载然后用浏览器打开 index.html，不需要装任何东西。
