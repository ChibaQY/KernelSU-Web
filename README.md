# KernelSU Web

闲着没事把 KernelSU Manager 的 UI 扒下来玩，结果越改越跑偏，打开里面全是原神。

你就当它是一个长得像 KernelSU 的云·原神启动器吧。

## 它能干嘛

其实啥也干不了。所有按钮都能点，但点了也不会有什么实质性的事情发生。

- 首页有两个主题皮肤可以切着玩
- 超级用户列表里永远只有云·原神一个应用
- 模块列表的开关拨着挺解压，虽然是假的
- 设置页改啥都不生效，但布局看着还挺唬人的
- 剩下几个页面凑数的，该有的都有

## 怎么用

下载下来用浏览器打开 index.html 就行，啥也不用装。
